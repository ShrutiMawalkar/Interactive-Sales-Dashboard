# Interactive-Sales-Dashboard
**Built an interactive Tableau dashboard to analyze sales trends, regional performance, and product-wise contribution.**
**Included key visualizations such as sales over time, region comparison, and a product tree map.**
Added filters for date range, region, and product category to enable dynamic analysis.
Implemented tooltips and drill-down interactions for deeper data exploration.
Delivered the project as a packaged Tableau workbook (.twbx) with an accompanying user guide.
